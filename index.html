Site Details

  1. Initially jQuery hides all page elements.
  2. jQuery slides in the red, white, and blue div's.
  3. jQuery is used to fade in the rest of the page.
  4. Bootstrap 3 is used for the navigation bar.
  5. Javascript is used to create the clock function.
  6. An external font alarmClock.ttf is used the clock display.
  7. jQuery slider widget is implemented.
  8. The slider value defaults to 50 and adjusts the opacity of the clock display.
  9. Bootstrap grid is used for responsive design structure.
  10. JQuery is used to "stick" navbar at top (medium and up devices only).
  11. JQuery is used to correct the link position for desktop viewing. Compensation is made for the desktop toolbar and fixed navigation bar.
  12. Print button prints the Resume only using @media rules.
  13. Additional Bootstrap components are used throughout, such as buttons, panels, glyphicons, carousel, and the "scrollspy" feature to indicate which section is active upon scrolling.

Hobbies

Geocaching

The concept of geocaching is that there are containers of various sizes "hidden in plain sight" throughout the world. There are likely many close by regardless of one's location. GPS coordinates are used to place yourself within the vicinity, but then the challenge is to actually find the container. Typically these are not virtual, there is an actual physical container to find. Sometimes there is a log to sign, or container of trinkets to trade, depending on the container size. This is a fun way to get outside, view scenery, and get some exercise. I have included some photos but I am not giving away all of the "secrets". For more information visit the link to the geocaching.com site.
