Site Details

  1. Bootstrap 3 is used for the navigation bar.
  2. Javascript is used to create the clock function.
  3. An external font alarmClock.ttf is used the clock display.
  4. JQuery slider widget is implemented.
  5. The slider value defaults to 50 and adjusts the opacity of the clock display.
  6. Bootstrap grid is used for responsive design structure.
  7. JQuery is used to "stick" navbar at top for all devices.
  8. JQuery code re-collapses expanded menu on mobile device after a link is clicked.
  9. JQuery is used to correct the link position for desktop viewing. Compensation is made for the desktop toolbar.
  10. Print button prints the Resume only using @media rules.
  11. Bootstrap components are used throughout, such as buttons, panels, glyphicons, carousel, and the "scrollspy".

Hobbies

Geocaching

The concept of geocaching is that there are containers of various sizes "hidden in plain sight" throughout the world. There are likely many close by regardless of one's location. GPS coordinates are used to place yourself within the vicinity, but then the challenge is to actually find the container. Typically these are not virtual, there is an actual physical container to find. Sometimes there is a log to sign, or container of trinkets to trade, depending on the container size.

This is a fun way to get outside, view scenery, and get some exercise. I have included some photos but I am not giving away all of the "secrets". I would encourage those interested to use their free app on your phone. For more information visit the link to the geocaching.com site.
