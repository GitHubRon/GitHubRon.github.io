School Project

My Franklin University curriculum included many programming languages such as HTML, CSS, Javascript, Java, Python, JQuery, MySQL, PHP, and Ruby. My capstone project utilizes HTML, CSS, Javascript, PHP, and MySQL. This project was not responsively designed, thus it is best viewed on a desktop. The project was for a fictitious automotive company named Lugnuts Automotive. It was completed by a two person team.

Lugnut Automotive

This project included a logo design, site design, project specification, interfacing with a provided database, creation of a shopping cart, creation of an order form, and form validation. It is hosted by Yahoo on the domain webweavin.com that I own. The link below will open in a new window.
